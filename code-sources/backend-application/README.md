# Intelligent Manager of Fantasy Premier League Game - Backend

## Requirements
For development and running the application, you will need Java 11 installed on your environment. You can also use newer versions.

### Java

To install Java, you can download free version from the website https://jdk.java.net/java-se-ri/11 and install it.

## Install and Start
The application uses Gradle Wrapper to download dependencies and run

    $ gradlew bootRun
