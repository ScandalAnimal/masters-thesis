# Intelligent Manager of Fantasy Premier League Game - Frontend

## Requirements

For development and running the application, you will need Node.js installed on your environment.
Also, you need Yarn package manager to load all project dependencies.
### Node

#### Installation on Windows
Grab the installer from the website: https://nodejs.org/en/download/ and install it
Please install at least version v10.15.3.

#### Installation on Linux

You should be able to install Node using Node Version Manager (NVM).
To install nvm, use:

    $ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

or grab latest install script from https://github.com/nvm-sh/nvm.
After installing the nvm, restart terminal, test if the nvm is installed and then use:

    $ nvm install node
    
to install the latest version of node.    

### Yarn

#### Installation on Windows

Grab the installer from the website https://classic.yarnpkg.com/en/docs/install and install it.

#### Installation on Linux

If you have downloaded Node, it is shipped with npm. Using npm, you can download Yarn:

    $ sudo npm install yarn -g

## Install

    $ yarn install

## Start
The application should run on default port 3000 using:

    $ yarn start
